# 🗺️ QGIS 制图项目

## 快速开始
打开 QGIS → 插件 → Python 控制台 → 运行脚本加载数据。

## 脚本路径
- D:\Reasonix\scripts\yangtze_qgis.py — 自动加载+配色
- D:\Reasonix\scripts\yangtze_qgis_style.py — 专业配色+布局+导出

## 在线底图（XYZ Tiles）
- ESRI卫星: https://services.arcgisonline.com/ArcGIS/rest/services/World_Imagery/MapServer/tile/{z}/{y}/{x}
- OSM: https://tile.openstreetmap.org/{z}/{x}/{y}.png

## 输出目录
- D:\Reasonix\output\

## 技能调用
在会话中输入 /qgis-mapping 直接进入制图模式。
